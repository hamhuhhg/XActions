Is this safe to use?

The browser scripts are completely safe - they just automate clicking. The SaaS version uses browser automation to avoid expensive API costs.

Will I get banned?

We include rate limiting to stay within acceptable use. However, unfollowing hundreds of people rapidly could trigger spam detection. Use responsibly!

Why use this instead of other tools?

  • 100% open source - see exactly what it does
  • Free browser scripts option
  • No expensive X API costs passed to you
  • Privacy-focused

Do you store my credentials?

Browser scripts: No, runs entirely in your browser.
SaaS: We securely store your session token. You can disconnect anytime.
